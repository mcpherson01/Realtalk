Introduction

Thank you for buying the Asgard - YouTube Alerts Builder.

If you like, take a look at our multi-social version of our Asgard plugin: Asgard - Social Media Alerts & Feeds WordPress Builder, where, among other things, you'll be able to link your social accounts to show Twitch streams, social feeds, video alerts, and much more! Click the image below to check it out!

The package includes the following:

  • Features

    • Builder application that allows to create alerter groups with dynamic and static content.

    • Profile management. You can save, load, edit and delete profiles. Activate a saved profile to display it in your active WordPress theme page.

    • Credentials management. Set your social network access keys at any time from the Credentials page.

    • Google fonts integration. You can choose to load any Google font to use on the alerts by specifying its name and weights in the builder.

  • Third Party plugins

    • Font Awesome integration, this means that from the builder, you can configure the alerts to use font awesome icons just by entering the icon class name.

  • Custom SVG icons

You can navigate the documentation sections by using the left pane menu, we recommend that you start with the Setup section in order to learn how to install and use the plugin.

How to Install

There are two ways to install the plugin. You can either install it manually or by using the WordPress upload functionality.

Upload Install

To install the plugin using this method you need to follow these steps:

  • Go to your WordPress administration panel.

  • Select the Plugins menu.

  • Click on the Add New button on the top left.

  • Click on the Upload Plugin button on the top left.

  • Click on Select File, select the asgard_alerts_yt_vX.X.X.zip plugin file that is inside the package Plugin/ directory.

  • Click on Install Now and the plugin installation will start.

  • The installation will finish and the plugin will now show in the Installed Plugins menu, from where you can activate it.

Manual Install

To install the plugin using this method you need to follow these steps:

  • Go to the package Plugin/ directory.

  • Unzip the asgard_alerts_yt_vX.X.X.zip file. This should create a new asgard_alerts_yt/ directory that contains the plugin files directly inside of it.

  • Copy the asgard_alerts_yt/ directory and paste it in your WordPress installation /wp-content/plugins/ directory.

  • Go to your WordPress admin panel and open the Plugins menu. The Asgard Youtube Alerts Builder plugin will appear on the list.

Quotas

Some social networks have limits (quotas) on how many requests you can make to them in a set period of time. For example you may be able to make "n" requests per minute, hours or day, after which every request will fail, so the social network won't reply to you with the information you requested but by telling you that your quota was met.
This limit refreshes per the same period of time, meaning that, for example, for an hour based limit, your quota resets every hour and the alerts will automatically display updated information.

Youtube

You can find more information about youtube limits here: https://developers.google.com/youtube/v3/getting-started#quota.

Creating a Profile

Profiles let you create and manage different alert groups. In order to show an alert group in your WordPress active theme page, you first need to create a profile and activate it.

To create a profile, you need to follow these steps:

  • Go to the plugin's Builder menu.

  • Click on the Add New Alert Group button.

  • Select a Social Network and click on the Next button.

  • Select an alert template and click on the Next button.

  • Click on the Start Customizing! button.

  • An alert group will be created with the selected template.

  • Assign a name to the profile. Click on the pencil icon on the top bar or where it says Enter profile name here... to edit the profile name.

  • After entering a profile name, click on the Save Alerts Group! button in the bottom left corner.
    This button lets you save new profiles as well as save edits done to any profile, just click it every time you want to save changes to the profile you are currently customizing in the builder.

  • That's it, you saved your first profile!. To see a list of created profiles and manage them, click on the My Profiles folder button on the bar.

  • Here, you will be able to activate, edit and delete created profiles. Refer to the other sections of the documentation for more information on the other available actions.

Activating a Profile

In order to show an alert group in your WordPress active theme page, you need to activate it.

To activate a profile, you need to follow these steps:

  • Click on the My Profiles folder button on the bar.

  • Turn on the Switch of the profile you want to activate.

  • The alert group will now appear in your WordPress active theme page!.

Editing a Profile

You can edit any profile you have created. Even if a profile is active, you can edit it, save the changes and they will update the alert group that shows in your WordPress theme active page automatically.

To edit a profile, you need to follow these steps:

  • Click on the My Profiles folder button on the bar to access your saved profiles.

  • Locate the profile you want to edit and click on the Edit button.

  • The profile will be loaded into the builder. Any unsaved progress you have done with the builder at the moment will be discarded, so make sure you save your progress before you start editing a profile.

  • Make any changes you want to the profile. When you are done editing, click on the Save Alerts Group! button on the bottom left.

  • The profile will be updated with the changes you made.

Deleting a Profile

You can delete any profile you have created.

To delete a profile, you need to follow these steps:

  • Click on the My Profiles folder button on the bar to access your saved profiles.

  • Locate the profile you want to delete and click on the Delete button.

  • A confirmation message will show. Click on the Delete button to delete the profile.

  • The profile will be deleted.

Managing Credentials

Some alerts have a Server tag, this means that their content is dynamic and is automatically filled with information from the social networks. The social networks, before providing the requested information, require some credentials to identify the user that is requesting the data.
We have a fully detailed step by step guide on how to get these credentials for the social network on the "Social Networks" section of the documentation.

To enter your credentials for the social networks you need to:

  • Go to the Credentials plugin menu.

  • Scroll down until you find the social network that you want to enter credentials for.

  • Enter your credentials.

  • Click on the Save Credentials! button that is on the top right.

  • That's it, your credentials are saved!.

Youtube

This section features a step by step explanation on how to get your youtube credentials.
Remember that after getting your credentials, you have to enter them in the Credentials menu.

Getting your credentials

In order to get your credentials you need to:

  • Register an app

  • Get API key

Register an app

  • Login to the Google account you want to register an app with.

  • Go to https://console.developers.google.com

  • Read and accept the Google Cloud Platform terms and conditions, select your country of residence and continue.

  • First, you will need to create a new Project, click on Create.

  • Enter your Project name and click Create.

  • Your new Project will be selected in the top dropdown. You will now need to Enable the Youtube API Service. To do this click on Enable APIS and Services.

  • Enter Youtube in the search field.

  • Select the Youtube Data API v3 from the results.

  • Click Enable to enable this service.

Get API key

  • After completing step 1, you will be presented with the following screen.

  • Click the Create Credentials button that is on the top right of the screen.

  • Select Youtube Data API v3 under Which API are you using?; Select Web browser (Javascript) under Where will you be calling the API from?; Select Public data under What data will you be accesing?

  • Click on the What credentials do i need? button

  • That's it!, you will be presented with your API Key, save it, you will have to enter it in the Credentials menu.
