Introduction

As you install "One Click Optimization" plugin for the first time, you get a admin section showing you a summary of your options and your system and admin settings.

If you are a complete WordPress beginner, I advise using optimization presets, then navigating through the options and see what each one does.

Installation Steps

1. Download this plugin from Codecanyon.

2. Unzip the one-click-optimization.zip

3. Upload all the files to your plugin directory. (your site directory / wp-content / plugins / ..)

4. Activate the plugin through the ‘Plugins’ menu in WordPress.

5. For customizing the plugin’s settings, click on "One Click Optimization" menu in WordPress admin menu > Settings.

The plugin will start working successfully.
