Get started with Yottie 1.3.0

Installation

1. Install Yottie WordPress Plugin yottie.zip to your site. Don't know how? Please learn here.

2. Go to Yottie page in WordPress admin menu.

3. Setup Yottie as you wish and copy a shortcode.

Use Yottie as WordPress Shortcode

You can use Yottie as WordPress shortcode and generate it using shortcode generator on Yottie page. Then copy-paste it into any page or post.

Example of shortcode:

[yottie 
  channel="https://www.youtube.com/user/starwars"
  width="auto"
  lang="en"
  header_layout="accent"
  content_columns="4"
  content_rows="2"
  video_layout="cinema"
]

Use Yottie as
Visual Composer Element

You can use Yottie as Visual Composer element and configure it during page building.

Options

Option Shortcode Param Default Value Description
Source
channel channel null Insert URL of a YouTube channel to display its information (Logo, banner, channel name, channel description, videos counter, subscribers counter and views counter) and videos from Uploads playlist in the plugin.
sourceGroups source_groups null Create custom groups of videos from any YouTube source (channels, playlists, videos). Name each group and specify unlimited number of YouTube sources in it. Please note: these sources will be displayed instead of videos from the specified channel above and will be presented in tabs. Use this option if you want to organize videos in your channel or create custom playlists of any videos.
cacheTime cache_time 3600 It defines how long a data from YouTube will be cached in a client side database IndexedDB. Set "0" to turn the cache off.

width width auto Plugin width (any CSS valid value: px, %, em, etc.). Set "auto" to make the plugin responsive.
Language

headerVisible header_visible true Turn on/off the header in the plugin.
headerLayout header_layout classic Choose one of three header layouts: classic (like in YouTube), accent (pay more attention to your channel), minimal (attract more attention to your playlist).

contentResponsive content_responsive null Accepts JSON string. It overwrites the grid view settings of the video gallery depending on different window sizes. Use the following format: { "600": { "columns": 2, "rows": 2, "gutter": 10 }}. It means when a window width would be less than or equal to 600px then the number of columns would be set to 2, the number of rows - to 2 and gutter - to 10px. Note: please use single quote ' with this option ([yottie responsive='{ "600": { "columns": 2, "rows": 2, "gutter": 10 }}'])

videoLayout video_layout classic Choose one of three video layouts: classic (video card with full information), cinema (pay more attention to video preview. Info displays on hover), horizontal (appropriate layout for displaying videos in 1-2 columns as a list).

videoPlayMode video_play_mode popup Choose the mode of watching videos: in popup ("popup"), directly in the video gallery ("inline"), or in a new browser tab right in YouTube ("youtube").
